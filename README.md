Deskripsi Proyek

Aplikasi ini merupakan sistem pengelolaan data menu makanan dan minuman bernama Menu Lezatku. Melalui aplikasi ini, pengguna dapat menambah, menampilkan, mengubah, dan menghapus data menu yang tersimpan di database. Selain itu, tersedia juga fitur pencarian menu berdasarkan nama atau kategori, sehingga memudahkan dalam pengelolaan data menu secara efisien.

---

Alasan Memilih Tema

Saya memilih tema ini karena pengelolaan data menu sering dijumpai pada berbagai usaha kuliner seperti kafe dan rumah makan. Tema ini juga mudah dipahami, namun tetap mencakup seluruh fungsi CRUD dengan penerapan database secara nyata. Selain itu, proyek ini dapat menjadi dasar untuk mengembangkan sistem pemesanan makanan di masa mendatang.
